## Overview

This system analyzes user preferences and movie data to suggest films you may enjoy. Built with versatility and scalability in mind, it can easily integrate new recommendation algorithms or data sources.

## Features

- **User-Based Collaborative Filtering**: Recommends movies by finding users with similar tastes.
- **Item-Based Collaborative Filtering**: Finds movies similar to those you liked.
- **Content-Based Recommendations**: Suggests movies based on genre, actors, and other features.
- **Hybrid Models**: Combines various approaches for improved accuracy.
- **Interactive User Interface**: Simple and intuitive for all users.

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aftb777/Movie-Recommendation-System.git
   cd Movie-Recommendation-System
   ```

2. **Set up a virtual environment (optional but recommended)**
   ```bash
   python -m venv venv
   source venv/bin/activate       # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```
